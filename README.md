# livima
Linux Virtualhost Manager via web GUI

# System requirements:
  - Operating System: Debian or Ubuntu

# Supports
  - Webserver:
    + NGINX
  - Database:
    + MariaDB
    + MySQL 5.7
    + MySQL 8
  - Languages:
    + PHP 5.6, 7.0, 7.1, 7.2, 7.3, 7.4, 8.0, 8.1

# Main Features:
- Create server name (Virtual host) for NGINX
- Create FTP account
- Manage domains
- Manage databases
- Manage PHP versions for your website
